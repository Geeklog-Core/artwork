The Geeklog "paperclip" logo was created by Simon Lord
who passed all the rights on to the Geeklog team.

We have since decided to release the logo under a
Creative Commons Attribution 3.0 license and made it
available through Wikimedia Commons:

http://commons.wikimedia.org/wiki/File:Geeklog-logo.png

This directory contains the same logo in PNG, PSD, and
EPS formats.


About the logo font

The font used in the logo is VAG Rounded:

http://en.wikipedia.org/wiki/VAG_Rounded

According to the Wikipedia article, VAG Rounded
was placed in the public domain.

For an alternative open source font, try Miso:

http://www.omkrets.se/typografi/
